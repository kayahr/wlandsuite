Wasteland Suite
===============

> [!NOTE]
> This project is no longer under active feature development. My current Wasteland-related work has moved to the TypeScript-based [wastelib](https://github.com/kayahr/wastelib) project. I still keep the Java Wasteland Suite working because I use it to create test assets, while wastelib currently focuses on reading game assets only.

The Wasteland Suite is a collection of command-line programs for converting
the game files of the famous game Wasteland (PC version) to standard format
files which can be processed by standard software.

Read more on the [Wasteland Suite Website](https://kayahr.github.io/wlandsuite/)
