Usage: extractmaps [OPTION]... OUTPUTDIR [INPUTDIR]
Extracts all the maps of wasteland as PNG images into a directory.

  -d, --debug    Shows stacktrace when an error occurs
  -h, --help     Display help and exit
  -V, --version  Display version and exit

The first parameter must be the name of the directory in which the PNGs
should be extracted. The program creates the necessary directory structure
automatically.

The second parameter specifies the wasteland directory. If the parameter is 
missing then the current directory is used.
